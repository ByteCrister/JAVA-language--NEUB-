## How to Use

1. Clone the repository:
    ```bash
    git clone https://github.com/ByteCrister/JAVA-language--NEUB-.git
    ```
2. Navigate to the desired folder based on the topic you want to explore.
3. Compile and run the Java files using:
    ```bash
    javac Filename.java
    java Filename
    ```
